O aplicativo denominado SAMP – Sistema de Apoio a Manutenção de Patrimônios,
foi desenvolvido utilizando a plataforma IntelXDK, que gera uma aplicação hibrida capaz de
rodar em diversas plataformas a partir do mesmo código fonte. Com isto conseguimos
abranger toda a população que frequenta a universidade.
Por ser uma aplicação móvel, permite que o usuário emita um alerta para o
responsável que determinado patrimônio está depreciado ou necessita de manutenção no exato
momento que percebe o problema, criando assim uma ponte que facilita a comunicação entre
o administrativo e os alunos ou demais funcionários.
